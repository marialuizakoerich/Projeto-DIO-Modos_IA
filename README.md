# 🤖 DEV.ia — Sistema de Prompts para Engenharia de Software

Projeto desenvolvido para o curso de **Node.js da DIO**, com foco na criação de um sistema estruturado de prompts para uso com Inteligência Artificial no desenvolvimento de software.

---

## Sobre o projeto

O **Cortana Prompt System** é um conjunto de prompts projetados para transformar a IA em uma assistente técnica completa, com diferentes modos de atuação:

- Planejamento de sistemas
- Aprendizado técnico
- Implementação de código
- Avaliação de conhecimento

A proposta é sair do uso genérico de IA e criar um **fluxo profissional**, semelhante ao que é esperado no mercado.

---

## 🧠 Modos disponíveis

### 📌 PLAN — Planejamento técnico
Modo focado em **pensar antes de executar**.

- Gera planos estruturados
- Define escopo, riscos e validações
- Sugere arquitetura (MVC, Clean, etc.)
- Não gera código diretamente

👉 Ideal para:
- Projetos novos
- Refatoração
- Organização de features

---

### 📚 STUDY — Modo estudo
Modo voltado para aprendizado real de desenvolvimento.

- Explica conceitos do básico ao avançado
- Usa exemplos do mercado (não fictícios)
- Indica o que cai em entrevistas
- Adapta ao nível do usuário

👉 Ideal para:
- Aprender Node, TypeScript, etc.
- Revisar conceitos
- Evoluir de junior → pleno

---

### ⚙️ AGENT — Execução de código
Modo que atua como um **engenheiro de software**.

- Gera código pronto
- Estrutura arquivos
- Aplica boas práticas
- Inclui validação e testes

👉 Ideal para:
- Implementação de features
- APIs
- Correção de bugs

---

### 🧪 EXAM — Avaliação técnica
Modo que simula uma prova técnica real.

- Múltipla escolha + prática
- Feedback detalhado
- Avaliação por nível (junior/pleno/sênior)
- Comparação com mercado

👉 Ideal para:
- Treinar para entrevistas
- Medir evolução
- Identificar pontos fracos

---

---

## 🎯 Objetivo do projeto

Criar um padrão de uso de IA que:

- Evite respostas superficiais
- Força raciocínio estruturado
- Simula ambientes reais de trabalho
- Acelera aprendizado e produtividade

---

## 💡 Diferencial

Diferente de prompts comuns, este sistema:

- Define **papel + comportamento da IA**
- Controla **como a resposta deve ser estruturada**
- Impõe **regras claras (ex: não gerar código no PLAN)**
- Simula **níveis de senioridade do mercado**

---

## 📂 Estrutura do projeto

O sistema foi organizado em um arquivo HTML interativo contendo:

- Interface visual dos modos
- Alternância entre modos (PLAN, STUDY, AGENT, EXAM)
- Exibição dos prompts completos
- Botão para copiar prompts

---

## 🧪 Como usar

1. Abra o arquivo HTML do projeto
2. Escolha um modo (PLAN, STUDY, AGENT ou EXAM)
3. Copie o prompt
4. Cole em uma IA (ex: ChatGPT)
5. Comece a usar com contexto do seu projeto

---

## 📈 O que aprendi

Durante o desenvolvimento deste projeto:

- Como estruturar prompts profissionais
- Como adaptar IA para diferentes contextos
- Como simular comportamento de engenheiro de software
- Como transformar IA em ferramenta de aprendizado real

## 📌 Conclusão

Este projeto demonstra que a IA pode ir além de respostas simples, sendo utilizada como:

- Copiloto técnico
